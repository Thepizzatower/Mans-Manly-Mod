# Man's Manly Mods

A BepInEx mod for **Ages of Conflict: World War Simulator** by **man_82**.

## Features

- **Flag map** (F8): every country rendered as its own flag stretched across its
  territory — detached lands like an Alaska get their own full flag
- **Curved country names** that follow the shape of the land, shrink to stay
  inside their borders, with custom fonts and colors
- **Focus view** (F9): one country in blue, the rest grey with configurable
  outlines — the Cold War documentary look
- **Search** (F10): find any country or city and jump the camera to it
- **War-front army numbers**, HOI4-style, sized by population
- **Timeline replay**: rewind and replay the whole run, draggable control bar
- **Population simulation**: cities and land feed a living population that
  drives the economy, taxes fund war chests, editable per-country from the
  nation panel
- **Smarter wars**: per-city combat efficiency, bigger armies push harder,
  momentum breaks stalemates, broke nations no longer insta-peace
- **Self-updating**: the mod checks this repo on startup and installs new
  versions automatically — install once, stay current

## Install

1. Install [BepInEx 5](https://github.com/BepInEx/BepInEx/releases) into the game folder
2. Grab the newest `MansManlyMods-vX.Y.Z.dll` from
   [Releases](https://github.com/Thepizzatower/Mans-Manly-Mod/releases) and drop
   it into `BepInEx/plugins/`
3. Play. Settings live in the game's own settings window under the mod tab
   (or press comma)

Updates after that are automatic.
